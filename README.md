# Interactive Greeting Page

A fun, interactive page that can be customized for any occasion (Valentine's Day, Birthdays, Anniversaries, Apologies, etc.). The "Yes" button grows bigger every time they click "No" — featuring falling hearts, cute GIFs, music, and playful toast messages. Built with HTML, CSS, and JavaScript.

**Live demo:** [https://kingsheartz.github.io/my-sweetheart](https://kingsheartz.github.io/my-sweetheart)

### Option 2: Fork It and Customize for Any Occasion

Want your own version you can customize? Follow these steps:

#### 1. Fork the Repo

- Go to [https://github.com/kingsheartz/my-sweetheart](https://github.com/kingsheartz/my-sweetheart)
- Click the **Fork** button in the top-right corner
- This creates a copy of the repo under your GitHub account

#### 2. Personalize It via `config.js`

You don't need to touch any HTML or core logic! 
- Simply open the `config.js` file and edit the text, messages, and GIFs to match your occasion (e.g., change it to "Will you forgive me?" or "Happy Birthday!").
- You can also point to different images or a new background song.

#### 3. Enable GitHub Pages

- In your forked repo, go to **Settings** (the gear icon tab).
- In the left sidebar, click **Pages**.
- Under **Build and deployment > Source**, select **GitHub Actions**.
- The included workflow will automatically build and deploy your customized site!

#### 4. Wait ~2 Minutes

- Your custom site will be live at:
  ```
  https://yourusername.github.io/my-sweetheart
  ```
  *(Replace `yourusername` with your actual GitHub username)*
